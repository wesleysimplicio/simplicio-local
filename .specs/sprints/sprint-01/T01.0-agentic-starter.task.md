# T01.0 - Completar Agentic Starter do repositorio

## Contexto

O repositorio ja contem o planejamento US4 V6 Apple Edition e a maior parte do
Agentic Starter, mas alguns arquivos de handoff e validacao estavam ausentes
ou removidos durante a adaptacao do scaffold. Antes de iniciar o runtime C++ da
Sprint 01, a base agentica precisa estar completa e verificavel.

## Acceptance Criteria

- [x] AC-1 - `INIT.md` existe e explica como continuar o trabalho neste repo.
- [x] AC-2 - `_BOOTSTRAP.md` existe e documenta o starter aplicado ao US4.
- [x] AC-3 - `.codex/config.toml` existe com configuracao Codex alinhada ao
  projeto.
- [x] AC-4 - `test/` existe com smoke tests do CLI `agentic-starter`, e
  `tests/e2e/` fica reservado para Playwright do produto.
- [x] AC-5 - `package.json` possui scripts `lint`, `test`, `test:cli` e
  `pack:dry` para validar o scaffold sem dependencias novas.
- [x] AC-6 - `npm test -- --coverage` gera `coverage/coverage-summary.json`
  com coverage acima de 80%.

## Out of scope

- Nao cria CMake/runtime skeleton de `T01.1`.
- Nao altera contratos C++.
- Nao adiciona dependencias npm, CMake, Homebrew, vcpkg ou MLX.
- Nao abre PR nem cria release.
